About Me

Hi! My name is Shravan Doda. I’m a 21 year old student pursuing my undergraduate degree in Computer Science from BITS Pilani, Goa Campus.

I love to code. I have a great interest in application development and system programming. I’m an avid open-source supporter. I like working on web technologies and operating systems. These days I’m working with Hydra Ecosystem where we are developing web tools based on linked data and semantic web. You can find my humble projects on Github.

Apart from programming, you’ll probably find me playing table tennis or video games, goofing around with my dog (his name is Toby) or listening to music. I also like to try and hack around with linux tools for improving my efficiency (I use Arch linux with i3-gaps as my window manager, Neovim as my default text editor).
